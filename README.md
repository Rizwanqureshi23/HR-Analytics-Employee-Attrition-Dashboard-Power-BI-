# HR-Analytics-Employee-Attrition-Dashboard-(Power-BI)


# Project Overview
This project focuses on HR Analytics and Employee Attrition Analysis using Microsoft Power BI.
The dashboard helps HR teams and management understand employee demographics, salary trends, attrition patterns,
and workforce health to support data-driven HR decisions.


# Business Problem

Organizations often struggle to:

* Monitor employee attrition effectively
* Understand key reasons behind employee turnover
* Analyze workforce demographics and salary trends
* Make proactive HR decisions

This dashboard solves these challenges by providing clear KPIs, trends, and interactive insights.


# Solution

* Designed an end-to-end **HR Analytics dashboard in Power BI** to monitor employee performance, workforce demographics, and attrition patterns.
* Created a centralized **Overview page** to track total employees, average salary, and attrition rate, enabling quick executive-level decision making.
* Built an **Attrition Analysis module** to identify attrition trends over time and analyze key reasons behind employee turnover.
* Implemented **interactive slicers, drill-down, and drill-through functionality** to allow dynamic and detailed employee-level analysis.
* Developed a **Date Table** and applied **DAX measures** to calculate HR KPIs such as attrition rate, active employees, and terminated employees.
* Utilized **Power Query** for data cleaning, transformation, and standardization to ensure high data accuracy and consistency.
* Enhanced dashboard usability by designing **custom backgrounds in PowerPoint** and implementing smooth page navigation.
* Enabled HR teams to **identify high-risk attrition segments** and take data-driven actions to improve employee retention.
